# Getting Started

This section covers everything you need to get Arawn running on your system.

## What You'll Learn

- **[Installation](installation.md)** — Build from source, configure dependencies
- **[Quick Start](quickstart.md)** — Your first chat, basic commands
- **[Configuration](configuration.md)** — Set up LLM backends and preferences

## Prerequisites

- **Rust** — 1.75+ with cargo
- **LLM Access** — API key for Anthropic, OpenAI, Groq, or local Ollama
- **SQLite** — Bundled via rusqlite (no external dependency)

## Typical Setup Flow

1. Clone and build the binary
2. Configure an LLM backend
3. Start the server or use CLI directly
4. Chat with memory persistence
